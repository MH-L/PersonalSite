Projects

Renju Deluxe

A Java swing GUI Gomoku client with single player, multi-player, network game functionalities. When playing in single player mode against computer, the player can choose between various difficulty levels as well as Sente/Gote. Online opponents are chosen based on player experience level, and all game logics are done on the server side to prevent cheating by using a hacked client.

TicTacToe Elite

An Android application as my first exercise for programming an AI, allowing players to play TicTacToe against computer or against another player on the Internet. Different difficulty levels are available for players so newbees can choose to play with easier levels.

Stock System

A website structured in HTML, CSS, Javascript, PHP and SQL, modeling a stock operating system with DB entities normalized to 3NF. Not only each transaction is recorded for analysis, the system also fetches latest stock data from google finance periodically.

Noti-Timer

An Android application which allows users to set multiple tasks that starts automatically after a certain period of time. In the initial release, playing music and sending instant messages are supported. Battery optimization is implemented to prevent excessive usage of battery.
